# Repeat

## Summary

Duplicates geometry according to a defined rule.

---

## Purpose

The Repeat node creates multiple copies of existing geometry. Through repeated application of simple rules, complex patterns can emerge from simple shapes.

---

## Computational Thinking Concepts

* Iteration
* Pattern Recognition
* Emergence

---

## Mathematical Principle

Repeated application of the same transformation generates increasingly complex structures.

---

## Inputs

Geometry.

---

## Outputs

Repeated geometry.

---

## Parameters

### Count

Number of repetitions.

### Offset

Distance between repeated elements.

### Direction

Direction in which geometry is repeated.

---

## Visualisation

A single object is progressively duplicated to form a complete pattern.

---
