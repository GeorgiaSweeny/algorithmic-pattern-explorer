# Grid

## Summary

Creates a regular arrangement of positions for subsequent computations.

---

## Purpose

The Grid node divides the workspace into evenly spaced cells. Many procedural algorithms use grids as an organisational structure for placing geometry or sampling values.

---

## Computational Thinking Concepts

* Spatial Reasoning
* Pattern Recognition
* Abstraction

---

## Mathematical Principle

A grid partitions two-dimensional space into a regular lattice of coordinates.

---

## Inputs

Workspace.

---

## Outputs

Grid structure.

---

## Parameters

### Rows

Number of horizontal divisions.

### Columns

Number of vertical divisions.

### Cell Size

Size of each grid cell.

### Spacing

Distance between neighbouring cells.

---

## Visualisation

The grid progressively appears across the workspace.
