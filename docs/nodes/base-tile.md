# Base Tile

## Summary

Defines a repeating geometric unit used for tessellation.

---

## Purpose

The Base Tile node provides the foundational shape for repeating pattern systems such as Escher-inspired tessellations.

---

## Computational Thinking Concepts

- Abstraction
- Pattern Recognition
- Spatial Reasoning

---

## Mathematical Principle

Space-filling geometry using repeated transformations.

---

## Inputs

Workspace.

---

## Outputs

Tile geometry.

---

## Parameters

### Shape Type

Initial tile geometry. For the Escher tessellation system, the available options are Square and Hexagon.

### Aspect Ratio

Controls tile proportions.

---

## Visualisation

A single tile is constructed and highlighted.

---

## Try Changing...

Modify aspect ratio and observe tiling distortion.
